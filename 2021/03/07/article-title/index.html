Self-Cure Net 阅读笔记

简单摘要

该网络（简称SCN）可以在面部表情识别中，抑制不确定性并防止深度网络过度拟合不确定的面部图像。
主要有两个方面：
（1）采用自注意力，对排名规则化以对训练样本加权
（2）对低排名样本重标记
简单来说，这个网络挑出那些标记错误的，或是模棱两可的图片，判断这些图片什么标签更合理，再丢回去进行识别。

主要模块：

（1）自注意力重要性加权(Self-Attention Importance Weighting)

首先当然需要知道哪些样本是有问题的，那便需要一些规则来判断，于是：通过CNN网络提取到一系列特征后（如F = [x1,x2,…,xn])将该FC层通过 $\alpha_i = \sigma(W_a^Tx_i)$得到一系列权重，权重高自然说明样本重要性高（或者说‘质量’高）。在原文提供的源码中，使用resnet18作为backbone network。
~~~
def init(self, pretrained = True, num_classes = 7, drop_rate = 0):
super(Res18Feature, self).init()
self.drop_rate = drop_rate
resnet = models.resnet18(pretrained)

    # self.feature = nn.Sequential(*list(resnet.children())[:-2]) # before avgpool
    self.features = nn.Sequential(*list(resnet.children())[:-1]) # after avgpool 512x1

    fc_in_dim = list(resnet.children())[-1].in_features # original fc layer's in dimention 512

    self.fc = nn.Linear(fc_in_dim, num_classes) # new fc layer 512x7
    self.alpha = nn.Sequential(nn.Linear(fc_in_dim, 1),nn.Sigmoid())

def forward(self, x):
    x = self.features(x)

    if self.drop_rate > 0:
        x =  nn.Dropout(self.drop_rate)(x)
    x = x.view(x.size(0), -1)

    attention_weights = self.alpha(x)
    out = attention_weights * self.fc(x)
    return attention_weights, out

>原文中损失函数采用对数加权交叉熵损失（WCE-Loss）：
 $\mathcal{L}_{WCE} = - {1\over N} \sum_{i=1}^N \log{\frac{e^{\alpha_iW_{yi}^Tx_i}}{\sum_{j=1}^Ce^{\alpha_iW_{j}^Tx_i}}}$ 

### （2）排序正则化（RankRegularization）
>已经知道样本的重要性，然后便要选出有‘质量问题’的样本。通过（1）得到样本权重$\alpha_i$($\alpha\in(0,1)$),然后对权重降序排列(权重高则说明样本有较高“重要性”)，分为比率为$\beta$的两组（高重要组：$M=\beta N$,低重要组：N-M），再分别取平均值$\alpha_H$和$\alpha_L$。
 $$ \alpha_H={\frac{1}{M}}\sum_{i=0}^M\alpha_i,\alpha_L={\frac{1}{N-M}}\sum_{M}^N\alpha_i $$
同时定义排序正则化损失（RR-Loss）：
 $$ \mathcal{L}_{RR} = max\{0,\delta_1 - (\alpha_H-\alpha_L)\} $$ 
其中$\delta_1$是一个超参数。将（1）与（2）中两损失函数以一定比率相加遍得总损失函数： $\mathcal{L}_all=(1-\gamma)\mathcal{L}_{RR}+\gamma\mathcal{L}_{WCE}$ 。

# Rank Regularization
        _, top_idx = torch.topk(attention_weights.squeeze(), tops)
        _, down_idx = torch.topk(attention_weights.squeeze(), batch_sz - tops, largest = False)

        high_group = attention_weights[top_idx]
        low_group = attention_weights[down_idx]
        high_mean = torch.mean(high_group)
        low_mean = torch.mean(low_group)
        # diff  = margin_1 - (high_mean - low_mean)
        diff  = low_mean - high_mean + margin_1

        if diff > 0:
            RR_loss = diff
        else:
            RR_loss = 0.0


### （3）重标记（Relabeling）：
>‘质量’合格的样本直接可以进行下一步，而低‘质量’的样本先扣下进行额外的‘加工’：换标签。不确定的样本有较低重要性，因此对低重要性组进行重标签：

$$
y' = 
\begin{cases}
    l_{max},\ \   if \ P_{max}-P_{gt}>\delta_2  \\
    l_{org},\ \   otherwise
\end{cases}
$$

若有最大预测的概率与给定标签概率之差大于阈值delta_2，则更换标签为该索引，否则保存原标签。重标签完成后遍可丢回训练集。

# Relabel samples
if i >= args.relabel_epoch:
    sm = torch.softmax(outputs, dim = 1)
    Pmax, predicted_labels = torch.max(sm, 1) # predictions
    Pgt = torch.gather(sm, 1, targets.view(-1,1)).squeeze() # retrieve predicted probabilities of targets
    true_or_false = Pmax - Pgt > margin_2
    update_idx = true_or_false.nonzero().squeeze() # get samples' index in this mini-batch where (Pmax - Pgt > margin_2)
    label_idx = indexes[update_idx] # get samples' index in train_loader
    relabels = predicted_labels[update_idx] # predictions where (Pmax - Pgt > margin_2)
    train_loader.dataset.label[label_idx.cpu().numpy()] = relabels.cpu().numpy() # relabel samples in train_loader

小结

SCN通过自注意力及排名正则化机制对“模糊”的样本进行重标签，从而降低了样本不确定性对训练深度神经网络的影响。同时，SCN的端到端训练特点，使其易于与各种CNN网络结合。
